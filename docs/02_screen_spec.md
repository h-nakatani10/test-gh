# 画面仕様書

## 1. タスク一覧画面

### 1.1. 概要
登録されているタスクを一覧で表示する。各タスクの編集・削除もこの画面から行う。

### 1.2. 画面レイアウト
-   画面上部に「新規タスク追加」ボタンを配置する。
-   タスクはテーブル形式で表示する。

### 1.3. 表示項目
| No. | 項目名     | 説明                                     |
| :-- | :--------- | :--------------------------------------- |
| 1   | ID         | タスクの一意なID                         |
| 2   | タイトル   | タスクの件名                             |
| 3   | 状態       | 「未着手」「進行中」「完了」のいずれか   |
| 4   | 操作       | 「編集」「削除」ボタンを配置             |
